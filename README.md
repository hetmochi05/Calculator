# 🧮 Simple Calculator

A sleek, responsive calculator built with vanilla HTML, CSS, and JavaScript — no libraries, no frameworks.

![Calculator Preview](preview.png)

---

## ✨ Features

- **Basic Operations** — Addition, Subtraction, Multiplication, Division
- **Advanced Input Handling** — Prevents operator spam, double decimals, and invalid expressions
- **Percentage Support** — `50%` automatically converts to `0.5`
- **Toggle Sign** — Flip any number between positive and negative with `+/-`
- **Calculation History** — Last 10 results saved and clickable to restore
- **LocalStorage** — History persists across page refreshes
- **Keyboard Support** — Use number keys, operators, `Enter`, `Backspace`, `Escape`
- **Haptic Feedback** — Vibration on mobile devices
- **Sound Effects** — Subtle click sound on button press
- **Dark / Light Mode** — Auto-detects system preference
- **Error Handling** — Shows `Error` with shake animation for invalid expressions
- **Responsive Design** — Works on mobile and desktop

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `0–9` | Enter number |
| `+ - * /` | Operators |
| `.` | Decimal point |
| `Enter` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` | Clear display |
